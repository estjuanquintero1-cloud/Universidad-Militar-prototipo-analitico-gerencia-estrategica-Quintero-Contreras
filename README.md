# Paso 1: Objetivo del Indicador

| Elemento | Descripción |
|-----------|-------------|
| Objetivo | Medir la proporción de clientes que rechazan la oferta de retoma debido a una diferencia significativa entre el valor esperado por el cliente y el valor ofertado por la compañía. |

# Paso 2: Factor Crítico

| Elemento | Descripción |
|-----------|-------------|
| Factor crítico | Gestionar adecuadamente las expectativas de precio de retoma para minimizar pérdidas de oportunidades comerciales. |

# Paso 3: Fórmula del Indicador

| Elemento | Descripción |
|-----------|-------------|
| Numerador | Número de clientes que rechazan la retoma cuando la diferencia entre expectativa y oferta supera el umbral definido. |
| Denominador | Total de clientes con oferta de retoma realizada. |
| Fórmula | (Numerador / Denominador) × 100 |

# Paso 4: Unidad de Medida

| Elemento | Descripción |
|-----------|-------------|
| Unidad | Porcentaje (%) |

# Paso 5: Fuentes de Información

| Elemento | Descripción |
|-----------|-------------|
| Fuente principal | Registro de oportunidades CRM |
| Fuente secundaria | Encuestas de cierre y seguimiento comercial |

# Paso 6: Seguimiento

| Elemento | Descripción |
|-----------|-------------|
| Frecuencia | Mensual |
| Responsable | Área Comercial |
| Meta | Reducir el porcentaje de rechazos por brecha de expectativa de precio |

**Condición de refutación:** Si el porcentaje de clientes que rechazan la oferta por diferencia de precio no disminuye después de implementar acciones de alineación de expectativas, la hipótesis se considera refutada.
